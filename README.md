# React URL Shortener by Yogi Ghorecha

A simple URL shortener built with React and Vite. It uses a Vite proxy to communicate with the TinyURL API.

## Installation

1. **Clone the repository**
   ```bash
   git clone [https://github.com/yogig/url-shortener-react.git](https://github.com/yogig/url-shortener-react.git)
   cd url-shortener-react

2. **Install dependencies**

    npm install

3. **Run the development server**

    npm run dev

4. **The app will be available at http://localhost:5173**

---
### Tech Stack
-   **React**
-   **Vite (with Proxy)**
-   **JavaScript**
